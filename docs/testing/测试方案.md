### 关键词聚合 (`api/explore`)

| 测试用例 | 输入 | 预期 |
|---------|------|------|
| 空对话列表 | [] | 空数组 |
| 单条对话 | [{keywords:["星空"]}] | ["星空"] |
| 权重衰减 | 最近的"星空"，较早的"大海" | "星空" 排在前面 |
| 去重 | 多条都有"星空" | 只出现一次，权重累加 |
| 上限 8 个 | 20 个不同关键词 | 只返回 top 8 |

```typescript
describe('aggregateKeywords', () => {
  test('时间衰减：最近的权重更高', () => {
    const convs = [
      { keywords: ['星空'], created_at: '2024-01-02' },  // 最近
      { keywords: ['大海'], created_at: '2024-01-01' },  // 较早
    ]
    const result = aggregateKeywords(convs)
    expect(result[0]).toBe('星空')
  })

  test('返回最多 8 个关键词', () => {
    const convs = Array.from({ length: 20 }, (_, i) => ({
      keywords: [`kw${i}`],
      created_at: new Date(Date.now() - i * 60000).toISOString()
    }))
    const result = aggregateKeywords(convs)
    expect(result.length).toBeLessThanOrEqual(8)
  })
})
```

### 进度计算 (`ExploreScene.tsx`)

| 测试用例 | 输入 | 预期 |
|---------|------|------|
| 刚开始 | now = started_at | progress: 0, stage: departure |
| 10% 进度 | 10% 时间过去 | progress: 10, stage: departure |
| 50% 进度 | 一半时间 | progress: 50, stage: journey |
| 80% 进度 | 80% 时间 | progress: 80, stage: discovery |
| 100% 进度 | 时间已过 | progress: 100, stage: return |
| 超过时间 | now > estimated_return | progress: 100 (clamped) |

```typescript
describe('calcProgress', () => {
  const makeExploration = (startOffset: number, duration: number) => ({
    started_at: new Date(Date.now() - startOffset).toISOString(),
    estimated_return: new Date(Date.now() - startOffset + duration).toISOString(),
  })

  test('进度 0% → departure', () => {
    const exp = makeExploration(0, 60000)  // 刚开始，总时长 1 分钟
    const { progress, stage } = calcProgress(exp as any)
    expect(progress).toBeCloseTo(0, 0)
    expect(stage).toBe('departure')
  })

  test('进度 50% → journey', () => {
    const exp = makeExploration(30000, 60000)  // 过了一半
    const { progress, stage } = calcProgress(exp as any)
    expect(progress).toBeCloseTo(50, 0)
    expect(stage).toBe('journey')
  })

  test('进度超过 100% 被钳制', () => {
    const exp = makeExploration(120000, 60000)  // 已超时
    const { progress } = calcProgress(exp as any)
    expect(progress).toBe(100)
  })
})
```

### (V2) 记忆提取 (`src/lib/ai/memory.ts`)

| 测试用例 | 输入 | 预期 |
|---------|------|------|
| 提取结构化记忆 | 包含明确事件的对话文本 | 返回 Memory 对象，含 topic/emotion/intent 字段 |
| 可分享 vs 私密分类 | "我喜欢看星星" vs "我和小明吵架了" | 前者 shareable=true，后者 shareable=false |
| 敏感信息脱敏 | 对话中包含真实姓名"张三" | 姓名被替换为脱敏占位符（如"某位朋友"） |
| 空对话无记忆 | "" 或极短文本 "嗯" | 返回空数组 |
| 多条记忆提取 | 长对话包含多个话题 | 返回多个 Memory 对象，各自独立 |

```typescript
describe('Memory Extraction (V2)', () => {
  test('从对话中提取结构化记忆', () => {
    const text = '今天我们聊了很久关于星空的话题，卡皮巴拉说想去看银河'
    const memories = extractMemories(text)
    expect(memories.length).toBeGreaterThan(0)
    expect(memories[0]).toHaveProperty('topic')
    expect(memories[0]).toHaveProperty('emotion')
    expect(memories[0]).toHaveProperty('intent')
  })

  test('正确分类 shareable vs private', () => {
    const shareableMemories = extractMemories('我喜欢在河边散步看日落')
    const privateMemories = extractMemories('我和小明吵架了心情很差')
    expect(shareableMemories[0].shareable).toBe(true)
    expect(privateMemories[0].shareable).toBe(false)
  })

  test('敏感信息脱敏处理', () => {
    const memories = extractMemories('我朋友张三今天带我去了公园')
    const topicText = JSON.stringify(memories)
    expect(topicText).not.toContain('张三')
  })

  test('空或极短对话返回空数组', () => {
    expect(extractMemories('')).toEqual([])
    expect(extractMemories('嗯')).toEqual([])
  })
})
```

### (V2) 记忆匹配 (`src/lib/sim/memory-match.ts`)

| 测试用例 | 输入 | 预期 |
|---------|------|------|
| topic_sim 计算 | 两组 topic 集合 | 返回 0-1 之间的相似度值 |
| emotion_sim 计算 | 两组 emotion 向量 | 返回 0-1 之间的相似度值 |
| 综合评分权重 | topic=0.8, emotion=0.6, intent=0.4 | score = 0.6×0.8 + 0.25×0.6 + 0.15×0.4 = 0.69 |
| 禁止自匹配 | 同一用户 ID | 返回空结果或跳过自身 |
| 冷启动无记忆 | memories=[] | 返回空匹配列表 |
| 完全不相关 | 无交集的 topic/emotion/intent | score 接近 0 |

```typescript
describe('Memory Matching (V2)', () => {
  test('topic_sim 正确计算', () => {
    const topicsA = ['星空', '大海', '音乐']
    const topicsB = ['星空', '音乐', '绘画']
    const sim = calcTopicSim(topicsA, topicsB)
    expect(sim).toBeGreaterThan(0)
    expect(sim).toBeLessThanOrEqual(1)
  })

  test('emotion_sim 正确计算', () => {
    const emotionsA = { happy: 0.8, calm: 0.6 }
    const emotionsB = { happy: 0.7, calm: 0.5 }
    const sim = calcEmotionSim(emotionsA, emotionsB)
    expect(sim).toBeGreaterThan(0)
    expect(sim).toBeLessThanOrEqual(1)
  })

  test('综合评分权重 0.6/0.25/0.15', () => {
    const score = calcMatchScore(
      { topicSim: 0.8, emotionSim: 0.6, intentSim: 0.4 }
    )
    expect(score).toBeCloseTo(0.69, 2)  // 0.6*0.8 + 0.25*0.6 + 0.15*0.4
  })

  test('禁止自匹配', () => {
    const userId = 'user-123'
    const matches = findMatches(userId, userId)
    expect(matches).toEqual([])
  })

  test('冷启动无记忆返回空列表', () => {
    const matches = findMatchesForMemories([])
    expect(matches).toEqual([])
  })
})
```
