# 生命周期

> 用户、卡皮巴拉、探索、对话会话的完整生命周期设计。

---

## 4.1 用户生命周期

```
┌─────────┐    注册     ┌──────────┐   命名卡皮巴拉   ┌──────────┐
│ 未注册用户│──────────→│ 新用户    │───────────────→│ 活跃用户  │
│         │           │ (无卡皮)  │                │          │
└─────────┘           └──────────┘                └────┬─────┘
                                                      │
                           ┌──────────────────────────┘
                           │
                           ▼
              ┌─────── 日常循环 ───────┐
              │                       │
              │  聊天 → 探索 → 收获    │
              │    ↑              │    │
              │    └──────────────┘    │
              │                       │
              └───────────────────────┘
```

**关键节点：**

| 阶段 | 触发 | 页面 | 数据变化 |
|------|------|------|---------|
| 注册 | 用户提交表单 | /register | auth.users + profiles |
| 创建卡皮巴拉 | 首次进入 /chat | /chat (setup视图) | capybaras 新记录 |
| 首次对话 | 发送第一条消息 | /chat | conversations 首条 |
| 首次探索 | 点击出发或对话触发 | /explore | explorations 首条 |
| 收获物品 | 探索完成 | /explore (return) | explorations.items_found |

## 4.2 卡皮巴拉生命周期

```
创建
  ↓
[home] 在家休息
  ↓ ← 探索完成/串门归来
  ├──→ 聊天 → 心情变化 + 记忆积累
  ├──→ 被触发探索意愿
  │      ↓
  │    [exploring] 外出探索
  │      ↓
  │    发现物品
  │      ↓
  │    时间到达 → 惰性完成
  │      ↓
  │    返回 [home]
  │
  └──→ 被触发串门（未实现）
         ↓
       [visiting]
         ↓
       返回 [home]
```

## 4.3 探索生命周期

```
                 idle
                  │
     用户点击出发 / 对话触发
                  │
                  ▼
        ┌─────────────────┐
        │   创建探索记录    │  POST /api/explore
        │  status: ongoing │
        │  capybara→exploring│
        └────────┬────────┘
                 │
     ┌───────── 时间流逝 ──────────┐
     │                             │
     │  前端每秒更新进度条           │
     │  progress = (now-start)/(end-start) │
     │                             │
     │  0-25%  → DepartureStage    │
     │  25-75% → JourneyStage     │
     │  75-95% → DiscoveryStage   │
     │  95%+   → ReturnStage      │
     │                             │
     └────────────┬────────────────┘
                  │
                  ▼
     estimated_return 时间到达
                  │
                  ▼
     用户打开页面 → GET /api/explore
                  │
                  ▼
        ┌─────────────────┐
        │   惰性完成检查    │
        │  status→completed │
        │  capybara→home   │
        │  completed_at=now │
        └────────┬────────┘
                 │
                 ▼
            归来动画
                 │
                 ▼
          用户确认收获 → 回到 idle
```

## 4.4 对话会话生命周期

```
用户进入 /chat
  │
  ├─ 无卡皮巴拉 → 显示命名界面 → POST /api/capybara → 创建
  │
  └─ 有卡皮巴拉 → 加载最近 50 条消息 → 显示聊天界面
       │
       ├─ 用户发送消息
       │    ↓
       │  [乐观更新] 立即显示用户消息
       │    ↓
       │  POST /api/chat → AI 处理
       │    ↓
       │  显示 AI 回复 + 更新心情
       │    ↓
       │  如果 want_to_explore → 显示探索提示卡
       │
       └─ 每 15 秒检查探索状态
            ↓
          如果探索完成 → 显示"卡皮巴拉回来了"提示
```
