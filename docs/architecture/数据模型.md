## 二、索引设计

### V1 索引（保留）

| 索引名 | 表 | 列 | 用途 |
|-------|---|---|------|
| `idx_conversations_user_time` | conversations | (user_id, created_at DESC) | 按时间获取对话历史 |
| `idx_explorations_user_status` | explorations | (user_id, status) | 查询进行中的探索 |
| `capybaras_owner_id_key` | capybaras | owner_id (UNIQUE) | 确保一人一只 |

### V2 索引

| 索引名 | 表 | 列 | 用途 |
|-------|---|---|------|
| `idx_memories_capybara` | memories | (capybara_id, created_at DESC) | 按卡皮巴拉获取记忆时间线 |
| `idx_memories_shareable` | memories | (is_shareable, category) | 全局匹配池查询可分享记忆 |
| `idx_memories_user` | memories | (user_id, is_shareable) | 用户记忆库管理 |
| `idx_travels_capybara_status` | travels | (capybara_id, status) | 查询进行中的旅行 |
| `idx_travels_user_time` | travels | (user_id, started_at DESC) | 用户旅行历史 |
| `idx_travels_location` | travels | (location_id) | 按地点聚合旅行 |
| `idx_journals_travel` | journals | (travel_id, day_number) | 按旅行获取每日手记 |
| `idx_locations_active_tier` | travel_locations | (is_active, tier) | 可用地点查询 |
| `idx_costume_items_slot` | costume_items | (slot, is_available) | 按槽位查询可用装扮 |

## 三、JSONB 字段结构

### V1 字段（保留）

**`capybaras.traits`** — 性格标签数组：
```json
["治愈", "淡定", "好奇", "友善"]
```

**`capybaras.memory`** — 长期记忆关键词（V1）：
```json
["失眠", "星空", "小时候的事", "想远行", "最近压力大"]
```

**`explorations.trigger_keywords`** — 触发此次探索的关键词：
```json
["星空", "神秘", "夜晚"]
```

**`explorations.items_found`** — 探索收获的物品：
```json
[
  {
    "name": "月光石",
    "description": "在溪边发现的会发光的石头",
    "category": "collectible",
    "rarity": "rare"
  },
  {
    "name": "蒲公英种子",
    "description": "轻轻一吹就飞走了",
    "category": "plant",
    "rarity": "common"
  }
]
```

### V2 字段

**`capybaras.equipped_costumes`** — 当前装扮槽位：
```json
{
  "head": "costume_item_id_1",
  "body": null,
  "tail": "costume_item_id_2",
  "accessory": null
}
```

**`memories` 表字段结构** — 结构化记忆记录：
```json
{
  "content": "用户提到小时候在乡下外婆家看萤火虫",
  "category": "childhood",
  "is_shareable": true,
  "tags": ["童年", "自然", "萤火虫", "乡村"],
  "source_conv_id": "conversation_uuid"
}
```

**`travels.costume_snapshot`** — 旅行时装扮快照：
```json
{
  "head": "costume_item_id_1",
  "body": null,
  "tail": "costume_item_id_2",
  "accessory": null
}
```

**`journals.panels`** — 手记面板（Webtoon 风格）：
```json
[
  {
    "order": 1,
    "image_prompt": "卡皮巴拉在京都岚山竹林中漫步，阳光从竹叶间洒下",
    "narrative": "清晨的竹林很安静。每一步都踩在落叶上，发出细碎的声响。",
    "mood": "peaceful"
  },
  {
    "order": 2,
    "image_prompt": "卡皮巴拉在竹林小径遇到另一只卡皮巴拉",
    "narrative": "拐角处，遇见了一只同样在发呆的家伙。它看起来也喜欢安静的地方。",
    "mood": "curious",
    "is_encounter": true
  }
]
```

**`travel_locations` 表字段结构**：
```json
{
  "name": "Arashiyama Bamboo Grove",
  "name_zh": "岚山竹林",
  "region": "Asia",
  "country": "Japan",
  "latitude": 35.0094,
  "longitude": 135.6670,
  "description": "京都西郊的竹林小径，阳光穿过层层竹叶",
  "tags": ["自然", "安静", "东亚", "竹林"],
  "tier": "mvp",
  "is_active": true
}
```
