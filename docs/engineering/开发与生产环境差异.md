# 开发与生产环境差异

> 控制文件：`src/lib/travel/timeConfig.ts` 中的 `TESTING_MODE` 开关

## 切换方式

```typescript
// src/lib/travel/timeConfig.ts
const TESTING_MODE = true   // 开发阶段
const TESTING_MODE = false  // 正式发布
```

发布前将 `TESTING_MODE` 改为 `false`，所有时间自动切换回真实刻度。无需修改其他文件。

---

## 时间刻度对照

| 概念 | 开发模式 (TESTING_MODE=true) | 生产模式 (TESTING_MODE=false) |
|------|------------------------------|-------------------------------|
| 1 "天" | **1 分钟** | 24 小时 |
| 旅行 1-5 天 | 1-5 分钟 | 1-5 天 |
| 休息 1-2 天 | 1-2 分钟 | 1-2 天 |
| 地点冷却 30 天 | 30 分钟 | 30 天 |

---

## UI 差异

| 功能 | 开发模式 | 生产模式 |
|------|----------|----------|
| 旅行倒计时 | 显示秒级实时倒计时 | 显示小时/分钟级倒计时 |
| 开发标签 | 页面显示 "🛠 开发模式：1天 = 1分钟" | 不显示 |
| 旅行状态轮询 | 每 10 秒 | 每 30 秒 |
| 休息倒计时 | 显示秒级实时倒计时 | 显示小时/分钟级倒计时 |

---

## 受影响的文件

| 文件 | 用途 |
|------|------|
| `src/lib/travel/timeConfig.ts` | 时间常量定义（唯一需要修改的文件） |
| `src/app/api/travel/route.ts` | 旅行发起/完成/休息计算 |
| `src/app/api/journal/route.ts` | 手记天数计算 |
| `src/app/api/chat/route.ts` | 聊天中旅行天数计算 |
| `src/app/travel/page.tsx` | 前端倒计时显示 + 轮询频率 |
