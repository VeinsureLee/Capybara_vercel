# 监控与可观测性

> 关键监控指标与日志策略。

---

## 12.1 关键监控指标

| 指标 | 来源 | 告警阈值 |
|------|------|---------|
| API 错误率 | Vercel Analytics | > 5% |
| AI 调用成功率 | 自定义日志 | < 80% |
| 平均响应时间 | Vercel Analytics | > 10s |
| 数据库连接数 | Supabase Dashboard | > 80% |
| 每日活跃用户 | Supabase/自定义 | 趋势监控 |

---

## 12.2 日志策略

```
API 路由日志（console.log → Vercel Logs）：
  [INFO]  POST /api/chat - user:{id} - provider:{name} - latency:{ms}
  [WARN]  AI provider {name} failed: {error}
  [ERROR] Database query failed: {error}

前端错误追踪（建议集成 Sentry）：
  - 未捕获异常
  - API 调用失败
  - 组件渲染错误
```
