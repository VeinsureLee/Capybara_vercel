# M8: 记忆模块 (`memory`)

> 从对话中提取结构化记忆，AI 自动分档 shareable/private，管理记忆库，为匹配系统提供数据基础。

---

## 模块概览

| 属性 | 说明 |
|------|------|
| 职责 | 记忆提取、分档分类、记忆库管理、匹配池供给 |
| 入口 | `/api/memory` (GET/PATCH/DELETE), `/api/chat` (自动提取) |
| 依赖 | AI Client, Supabase |
| 核心 | 对话 → AI 提取记忆 → 分档 → 存储 → 匹配池 |

---

## 记忆提取流程

```
用户消息
  ↓
[1] AI 记忆分类 Prompt（memory classify prompt）
  ↓
[2] 判断 has_memory?
  ├── false → 无记忆提取，流程结束
  └── true  ↓
[3] 提取结构化字段
     - topic：记忆主题
     - summary：简要描述
     - emotion：关联情绪
     - shareable：是否可分享（true/false）
  ↓
[4] 存储至 memories 表
  ↓
[5] shareable=true → 进入匹配池
    shareable=false → 仅用户可见
```

## AI 分档规则

### Private（私密）候选

- 家人具体信息（姓名、职业、关系细节）
- 健康/疾病
- 财务（债务、收入、具体数额）
- 法律纠纷
- 亲密关系细节
- 未成年人相关
- 用户明确说"别跟别人说"

### Shareable（可分享）候选

- 偏好、兴趣
- 一般情绪状态
- 童年经历主题（不含身份信息）
- 态度、观点
- 生活方式描述

### 敏感话题特殊处理

- 自杀/自伤 → "crisis" 类别，引导专业资源，永不进匹配池
- 暴力/犯罪 → 不跟进
- 医疗/法律咨询 → 明确"帮不了你"，引导专业渠道

## 卡皮的记忆意愿表达（双面机制）

| 后台（隐私系统） | 用户视角（卡皮表达） |
|---|---|
| AI 打标 shareable=true | "这件事挺有意思的，我想记着" |
| AI 打标 shareable=false | "这个就你我知道" / "我不跟别人说" |
| 敏感话题 | 表达关心，不追问、不转发 |

---

## 记忆库 UI（"卡皮的小本子"）

- 时间线展示所有记忆
- 每条右上角 🔓(shareable) / 🔒(private) 切换
- 可按分类筛选
- 每条可删除（同时从匹配池清除）
- 视觉：温暖的小本子风格，非后台管理页

## 数据结构

```typescript
interface Memory {
  id: string                          // UUID
  user_id: string                     // FK → auth.users
  capybara_id: string                 // FK → capybaras
  topic: string                       // 记忆主题
  summary: string                     // 简要描述
  emotion: string                     // 关联情绪
  shareable: boolean                  // 是否可分享
  sensitive_category: string | null   // 'crisis' | 'violence' | null
  source_conversation_id: string      // FK → conversations
  created_at: string
  updated_at: string
}
```
